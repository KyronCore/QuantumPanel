Terminal

                  
Terminal-style admin section for host access guidance.

                


                

                  Direct browser-to-host shell access is intentionally not wired here. The safe path is standard SSH to the Debian host.
                


                

                  
$ ssh KyronX@192.168.1.82

                  
Use this from your own machine after SSH is enabled on Debian.

                  
Later we can integrate a hardened gateway approach, but not a raw web shell.

                


                

                  
Terminal Plan

                  

                    
1. Install and enable OpenSSH on Debian

                    
2. Use key-based login

                    
3. Restrict panel users from direct shell unless needed

                    
4. Keep terminal outside the web app for now
